# Five lesson topics

Whatever module you start, it's always 5 sequential lessons gated by an 80% pass threshold.

| Topic | Source | When to use |
|---|---|---|
| 📝 **Code** | Active selection or current file | Quizzing on code an agent just wrote |
| 🏗️ **Infrastructure** | Build & config files | Understanding the build pipeline |
| 🛠️ **Tools** | Dependencies and scripts | Knowing what each library does |
| 🧱 **Architecture** | Project directory tree | Module boundaries |
| 🔐 **Security** | Active file or representative source | Spotting injection vectors, validation gaps |

Lessons render as a Duolingo-style path. Pulsing gold node = available; lock = earn it; check = completed.

Pass each with ≥ 80% to unlock the next.
