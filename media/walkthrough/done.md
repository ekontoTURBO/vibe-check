# You're all set 🎓

## What to do next

1. Close this walkthrough
2. Open the **🎓 sidebar** in the activity bar
3. Let an AI agent insert some code into your editor — Vibe Check spawns a quiz automatically

## Useful commands (Ctrl/Cmd+Shift+P)

- `Vibe Check: Configure Provider` — change LLM backend, swap models, rotate keys
- `Vibe Check: New Module...` — explicitly pick a topic
- `Vibe Check: Quiz Me On Selection` — generate a module from selected code
- `Vibe Check: Start Due Review` — daily review queue
- `Vibe Check: Switch Track` — beginner / intermediate / expert
- `Vibe Check: Reset Progress` — wipe everything and start fresh

## Power tips

- **Click backtick code references in question prompts** — they jump to the source in your editor
- **Wrong answer? Click `? WHY`** for a personalized LLM explanation that calls out *your specific mistake*
- **Pulse Observer auto-fires** on AI insertions ≥5 lines or ≥200 chars — toggle off via `vibeCheck.autoQuiz`
