# Three difficulty tracks

Pick the level that matches your current goal. Each track has its own progress, XP, streak, and review queue. Switch any time.

## 🟢 Beginner — 5 XP / correct
Recognition and recall.
> *"What does `Map.get(key)` return when the key is missing?"*

## 🟡 Intermediate — 10 XP / correct
Applied logic; predict outputs; reorder code.
> *"Trace the flow: what does this `async function` return when `fetch` rejects?"*

## 🟣 Expert — 20 XP / correct
Architecture, edge cases, trade-offs.
> *"Under what concurrency pattern does this lock-free queue lose updates?"*

Switch with **`Vibe Check: Switch Track`** any time.
