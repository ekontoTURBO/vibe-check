# Pick your AI provider

| Provider | Setup | Best for |
|---|---|---|
| **Copilot** (VS Code only) | Sign into Copilot — done | Free if you have Copilot |
| **Anthropic** Claude | Get a key → paste it in the wizard | Best educational explanations |
| **Gemini** | Get a key → paste it in the wizard | Cheapest at scale |
| **OpenAI** | Get a key → paste it in the wizard | Most familiar |
| **OpenRouter** | Get a key → paste it in the wizard | One key, 100+ models |

> ⚠️ Antigravity, Cursor, Windsurf, and VSCodium don't currently expose their built-in AI to extensions. You'll need a direct provider key — the wizard makes this a 60-second process.

API keys live in encrypted SecretStorage. Never in `settings.json`, never synced to other devices.
