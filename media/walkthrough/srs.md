# Spaced repetition is the magic

Cramming once doesn't make code stick. **Daily review does.**

Vibe Check uses the [Free Spaced Repetition Scheduler (FSRS)](https://github.com/open-spaced-repetition/ts-fsrs) — the same algorithm that powers modern Anki — to schedule each question for review at the moment you'd be about to forget it.

- ✅ Got it right? Card waits longer before reviewing again
- ❌ Got it wrong? Card reschedules sooner
- 🔥 Daily streak burns when you complete at least one review per day
- 🎯 Daily ring fills as you accumulate XP toward a 50 XP daily goal

Click **↻ START DUE REVIEW** in the sidebar whenever the badge shows due cards. 5-10 minutes a day is enough.
